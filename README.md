# defi-staking-dapp
Decentralized Web3 staking platform built on EVM-compatible blockchain. Users can connect wallets, stake tokens, earn rewards, and interact with smart contracts securely. Built with React, Solidity, and Ethers.js with focus on security and scalability.

## ✨ Features

- 🔐 Wallet Connection (MetaMask / WalletConnect)
- 💰 Token Staking
- 🎁 Reward Distribution System
- ⛽ Gas Optimized Smart Contract
- 📊 Real-time On-chain Data## 🛠 Tech Stack

- Frontend: React / Next.js
- Smart Contract: Solidity
- Web3 Library: Ethers.js
- Framework: Hardhat
- Storage: IPFS## 📦 Installation

git clone https://github.com/Arifin036/defi-staking-dapp.git
cd defi-staking-dapp
npm install
npm run dev## 🗺 Roadmap
